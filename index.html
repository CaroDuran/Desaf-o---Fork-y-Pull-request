Subir Trabajos a GitHub *********************** Previamente, ya debes entrar a tu sesion en github.com Pasos del 8 al 12 por la terminal, de donde esta el trabajo. git init (Iniciar Git) git status (Ver los archivos que no han sido subidos) git add --all git status (verificar si los archivos se han sido subido, se colocan de color verde) git commit -m "Colocar un comentario relativo al trabajo" Pasos del 16 al 19 en el navegador (pasos 23 y 24 en el navegador solo para copiarlos) Ir a github.com en el navegador New repository (boton new en color verde que esta a la izquierda) Colocar nombre del repositorio (Repository name) Create Repository Pasos del 23 al 24 pegar (lo que copiaste en el navegador) en el terminal Copiar la primera linea de ...or push y pegarla en la terminal Cambiar la ultima palabra (main) por master de la tercera linea de ...or push y despues copiarla y pegarla en la terminal (git push -u origin master)
